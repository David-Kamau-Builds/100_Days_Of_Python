## Getting Started
1. Clone the repository
2. Switch to the desired week branch: `git checkout Week_1`
3. Navigate to the day folder you want to work on
4. Run the Python files and follow along with the README

## Challenge Goals
- Build consistent coding habits
- Learn Python fundamentals through practical projects
- Create a portfolio of 100 Python programs
- Document learning progress and insights
