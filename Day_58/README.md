# Day 58: Web Design with Bootstrap

## Project Overview
A web application built with Flask that serves a responsive landing page designed with the Bootstrap framework. The project focuses on integrating front-end libraries with a Flask backend to create visually appealing and mobile-friendly interfaces.

## Key Concepts Learned
- **Bootstrap Framework**: Implementing the popular CSS framework for responsive design
- **Flask Setup**: Configuring a basic Flask application to serve HTML templates
- **Responsive Layouts**: Using the Bootstrap Grid System (Containers, Rows, Columns)
- **Component Usage**: Implementing pre-styled components like Navbars, Buttons, and Cards
- **External Assets**: Linking CSS stylesheets and JavaScript bundles via CDN

## Technical Skills
- Flask application initialization and route definition
- HTML5 structure with Bootstrap class integration
- CSS styling overrides and customization
- Rendering templates using `render_template`

## Features
- **Home Page**: Serves the main landing page (`index.html`) styled with Bootstrap.
- **Responsive Design**: Layout adjusts automatically for mobile, tablet, and desktop screens.

## Project Structure
- `server.py` - Main Flask application that serves the landing page
- `templates/` - Directory for HTML template files
  - `index.html` - Main landing page structure using Bootstrap classes
