# Day 8 - Caesar Cipher

## Overview
Day 8 of the 100 Days of Python challenge - a Caesar Cipher encoder/decoder with function parameters and modular programming.

## Project Features
- Encode and decode messages using Caesar cipher
- Custom shift amounts with wraparound handling
- Preserves non-alphabetic characters (spaces, punctuation)
- Continuous operation with restart option
- ASCII art logo integration

## Key Concepts
- **Function Parameters**: Passing multiple arguments to functions
- **Modulo Operation**: Handling alphabet wraparound
- **String Manipulation**: Character-by-character processing
- **Input Validation**: Handling different user inputs
- **Module Import**: Using external art module

## Notes
Introduces function parameters and arguments, demonstrating how to create reusable functions. The modulo operation ensures the cipher works with any shift amount, and the program handles edge cases like spaces and punctuation.
