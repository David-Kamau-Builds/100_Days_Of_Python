# Day 44: Intermediate CSS - Motivational Poster

## Project Overview
A responsive motivational poster webpage showcasing intermediate CSS techniques including flexbox layouts, responsive design, hover effects, and modern typography using Google Fonts.

## Key Concepts Learned
- **Flexbox Layout**: Centering content both horizontally and vertically using flexbox
- **Responsive Typography**: Using `clamp()` function for fluid, responsive font sizes
- **CSS Transitions**: Smooth hover effects with transform and transition properties
- **Box Model Reset**: Universal selector reset for consistent styling across browsers
- **Google Fonts Integration**: Loading and implementing custom web fonts
- **Responsive Images**: Making images scale properly across different screen sizes

## Technical Skills
- Flexbox for layout and alignment
- CSS `clamp()` function for responsive typography
- CSS transitions and transforms
- Media queries for responsive design
- Box-shadow and text-shadow effects
- CSS custom properties and modern selectors

## Features
- Fully responsive motivational poster design
- Smooth hover animation on image
- Fluid typography that scales with viewport
- Professional styling with custom fonts
- Mobile-optimized layout and spacing
- Elegant border and shadow effects
