# Day 9 - Secret Auction Program

## Overview
Day 9 of the 100 Days of Python challenge - a secret auction bidding platform using dictionaries, error handling, and cross-platform screen clearing.

## Program Features
- Multiple bidder support with input validation
- Cross-platform screen clearing (Windows/Mac/Linux)
- Error handling for invalid bid inputs
- Automatic winner determination
- Highest bid tracking
- ASCII art logo integration
- Edge case handling (no bids placed)

## Key Concepts
- **Dictionaries**: Storing key-value pairs (name: bid amount)
- **Dictionary Methods**: `.values()`, `.items()`
- **Built-in Functions**: `max()` for finding highest value
- **Error Handling**: Try/except blocks for input validation
- **OS Module**: Cross-platform screen clearing
- **Data Iteration**: Looping through dictionary items
