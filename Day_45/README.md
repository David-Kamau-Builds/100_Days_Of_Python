# Day 45: Web Scraping with Beautiful Soup

## Project Overview
Web scraping projects using Beautiful Soup library to extract and parse data from websites, including movie rankings and news articles with data processing and JSON output.

## Key Concepts Learned
- **Beautiful Soup Library**: HTML parsing and element selection using Beautiful Soup
- **Web Scraping Fundamentals**: Making HTTP requests and handling responses
- **CSS Selectors**: Using CSS selectors to target specific HTML elements
- **Data Extraction**: Extracting text content and attributes from HTML elements
- **Regular Expressions**: Using regex for text pattern matching and cleaning
- **JSON Data Handling**: Converting scraped data to JSON format for storage

## Technical Skills
- Beautiful Soup HTML parsing and navigation
- HTTP requests with the requests library
- CSS selector syntax for element targeting
- Regular expressions for data cleaning
- JSON file operations and data serialization
- Data sorting and manipulation techniques

## Features
- **Movie Scraper**: Extracts Empire's 100 Greatest Movies list with rankings
- **News Scraper**: Scrapes Hacker News for top stories with scores
- **HTML Parser**: Basic HTML parsing and element extraction practice
- **Data Processing**: Cleans and formats scraped data
- **JSON Output**: Saves structured data in JSON format
- **Sorting Algorithms**: Sorts data by rankings and scores

## Files
- `main.py` - Empire movie list scraper with ranking extraction
- `news.py` - Hacker News scraper for top stories
- `movies.json` - Extracted movie data in JSON format
- `getting_started/main.py` - Basic Beautiful Soup HTML parsing examples
- `getting_started/website.html` - Sample HTML file for parsing practice
