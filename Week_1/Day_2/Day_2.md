# Day 2 - Tip Calculator

## Overview
Day 2 of the 100 Days of Python challenge - a bill calculator that splits the total with tip among multiple people.

## Tasks
- [x] Get bill amount from user
- [x] Get tip percentage from user
- [x] Get number of people to split bill
- [x] Calculate tip amount
- [x] Calculate total per person
- [x] Display result rounded to 2 decimal places

## Notes
Tip calculator that takes user input for bill amount, tip percentage, and number of people, then calculates how much each person should pay including tip.
